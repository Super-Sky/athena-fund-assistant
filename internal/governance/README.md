# Financial Governance Module

`governance` is the deterministic product-level gate for financial decision output.

`governance` 是金融决策输出的确定性产品级治理门。

## File Index

- `gate.go`
  - Evaluates source/freshness disclosures, option shape, percentage derivation, and prohibited financial language.
  - 评估来源/新鲜度披露、方案结构、仓位比例依据和禁止的金融措辞。
- `gate_test.go`
  - Covers allowed, flagged, and blocked financial output states.
  - 覆盖允许、标记和阻断三类金融输出状态。
